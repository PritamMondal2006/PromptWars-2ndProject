## 📌 About the Project

**BengalPolls** is a web application that empowers West Bengal voters with transparent, accessible information about election candidates and historical results. Users can browse candidates by constituency, submit public reviews, and explore past Lok Sabha and Vidhan Sabha election outcomes — all in one place.

This project was built as a submission for **PromptWars: Virtual** — a bi-weekly AI hackathon by **Google × Hack2Skill** — using **Google Antigravity** (intent-driven / vibe coding) instead of traditional manual coding.


## ✨ Features

- 🏘️ **Constituency Browser** — Select your constituency and instantly see the candidates running in your area
- 👤 **Candidate Profiles** — View detailed information about each candidate including their party affiliation
- ⭐ **Public Reviews & Ratings** — Leave and read community reviews on candidates
- 📊 **Historical Election Results** — Explore past results for:
  - 2021 West Bengal Assembly Elections
  - 2019 Lok Sabha Elections
- 📱 **Responsive Design** — Works seamlessly on desktop and mobile
- ⚖️ **Strictly Neutral** — The platform does not endorse any party or candidate; all reviews reflect user opinions only
